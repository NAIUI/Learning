总共有 `10<sup>5</sup>` 个基因值，每个基因值都用 **闭区间** `[1, 10<sup>5</sup>]` 中的一个整数表示。给你一个下标从 **0** 开始的整数数组 `nums` ，其中 `nums[i]` 是节点 `i` 的基因值，且基因值 **互不相同** 。

请你返回一个数组 `ans` ，长度为 `n` ，其中 `ans[i]` 是以节点 `i` 为根的子树内 **缺失** 的 **最小** 基因值。

节点 `x` 为根的 **子树 **包含节点 `x` 和它所有的 **后代** 节点。
